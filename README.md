💬 Chatroom App
A real‑time chatroom application built with Node.js, Express, and Socket.IO. Users can create private chatrooms or group chatrooms and communicate instantly.

🚀 Features
Private Chatrooms – One‑on‑one private conversations

Group Chatrooms – Multiple users can join and chat together

Real‑time Messaging – Powered by Socket.IO for instant message delivery

Room Creation – Users can create new rooms on the fly
